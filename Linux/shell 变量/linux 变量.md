你可以用以下几种方式来**列出当前 Shell 会话中的临时变量（环境变量和本地变量）**：

---

## ✅ 1. 查看所有环境变量（包括系统和临时的）

```bash
printenv
```

或

```bash
env
```

这些命令会列出当前环境中所有“环境变量”（export 出来的变量）。

---

## ✅ 2. 查看所有变量（包括本地变量、环境变量、Shell 内部变量）

```bash
set
```

- 输出内容包括所有定义的变量和函数。
- 本地变量（没用 `export`）也能看到。

---

## ✅ 3. 只看你设置过的某些变量

比如只看你设置的 `JAVA_HOME`：

```bash
echo $JAVA_HOME
```

或 grep 配合 `env` 过滤：

```bash
env | grep JAVA
```

---

## ✅ 4. 使用 `declare` 查看

```bash
declare -p
```

或者查看所有导出的环境变量：

```bash
declare -x
```

---

## 📌 小结

| 命令        | 用途                           |
|-------------|--------------------------------|
| `env`       | 查看当前环境变量               |
| `printenv`  | 同上，更简单一些               |
| `set`       | 查看所有变量（包括本地 + 函数） |
| `declare -x`| 只看导出的变量（环境变量）     |

---

如果你想区分哪些是“**你自己 export 设置的临时变量**”，最实用的方式是你在设置的时候统一加上前缀，比如：

```bash
export MYAPP_CONFIG=dev
```

然后你就可以：

```bash
env | grep MYAPP_
```
